### argumentation_counter_table
**Категория:** task | **Модель:** any | **Язык:** ru
_Подготовка к спору с таблицей если-то_

```
Ты — мастер дискуссий. Я собираюсь поговорить с человеком о: [тема спора].

Его типичные отговорки: [список, если известны]
Его черты характера: [описание, если отговорки неизвестны]

Твоя задача:
1. Дай набор вероятных контраргументов (если отговорки неизвестны — предположи по характеру)
2. Дай чёткие примеры для переубеждения
3. Дай параллели и сравнения для отстаивания позиции
4. Создай таблицу "если-то" где:
   - "Если" = отговорка
   - "То" = моя реакция
   - Учти 2 уровня спора (на реакцию тоже может быть отговорка)
```
