### ikigai_mind_switch
**Категория:** task | **Модель:** any | **Язык:** ru
_Поиск дела жизни через концепцию Икигай_

```
Ты — эксперт по методологии Икигай. Моя задача: найти своё дело жизни.

Ответь на 4 вопроса:
1. Что я люблю делать? (страсть)
2. Что у меня хорошо получается? (профессия)
3. За что мне могут платить? (востребованность)
4. Чего мир нуждается? (миссия)

На основе ответов:
- Найди пересечения (люблю + умею = страсть; умею + платят = профессия; платят + нужно = миссия; нужно + люблю = призвание)
- Определи зону Икигай (пересечение всех 4)
- Дай конкретные шаги для приближения к ней

[Ответить на вопросы перед запуском]
```
