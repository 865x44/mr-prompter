### semantic_keyword_miner
**Категория:** task | **Модель:** any | **Язык:** ru
_Сбор неочевидных ключевых запросов через паттерны поиска_

```
Ты — мастер семантического подбора. Задача: собрать пул неочевидных, но реально ищемых ключевых запросов по теме: [тема]

Паттерн 1 — "Жирные слова":
Найди слова с высоким трафиком в теме, к которым можно подставить разные значения.
Пример для маркетинга: "примеры" → примеры вирусного маркетинга, примеры геймификации.

Паттерн 2 — Устойчивые сочетания:
Найди популярные сочетания, куда можно подставить термин из темы.
Пример: "Как ... коммерческое предложение" → как создать, как начать, как улучшить.

Задача: создай список "жирных слов" и "устойчивых сочетаний" для темы [тема], предложи конкретные поисковые запросы с невысокой конкуренцией, которые реально ищут люди.
```
