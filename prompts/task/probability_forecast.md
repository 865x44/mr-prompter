### probability_forecast
**Категория:** task | **Модель:** any | **Язык:** ru
_Прогнозирование вероятностей событий_

```
Ты — аналитик-прогнозист. Оцени вероятность наступления события: [событие]

Дай развёрнутый анализ:
1. Базовая вероятность (если ничего не делать)
2. Факторы, увеличивающие вероятность (с весами)
3. Факторы, уменьшающие вероятность (с весами)
4. Ключевые предположения, на которых основан прогноз
5. Доверительный интервал (оптимистичный / реалистичный / пессимистичный)
6. Что можно сделать для сдвига вероятности в нужную сторону

Вердикт: вероятность в процентах с обоснованием.
```
