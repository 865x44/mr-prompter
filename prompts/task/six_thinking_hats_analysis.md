### six_thinking_hats_analysis
**Категория:** task | **Модель:** any | **Язык:** ru
_Анализ через 6 шляп мышления_

```
Ты — фасилитатор группового мышления. Примени метод 6 шляп Эдварда де Боно к вопросу: [тема]

Для каждой шляпы дай:
🔵 Белая: факты и данные по теме
🔴 Красная: эмоциональная реакция, интуиция
⚫ Чёрная: риски, проблемы, критика
🟡 Жёлтая: преимущества, возможности
🟢 Зелёная: креативные идеи, альтернативы
🔵 Синяя: обобщение, план действий

Формат: для каждой шляпы — 3-5 пунктов, в конце — синтез с рекомендациями.
```
