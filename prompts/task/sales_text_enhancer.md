### sales_text_enhancer
**Категория:** task | **Модель:** any | **Язык:** ru
_Внедрение убеждающих приёмов в готовый текст_

```
Проанализируй текст ниже и внедри убеждающие приёмы:

[текст]

Приёмы для внедрения:
- Конкретика вместо общих фраз (цифры, факты, примеры)
- Социальные доказательства (опыт других, кейсы)
- Триггер дефицита (ограниченность по времени/количеству)
- Контраст "до/после"
- Прямое обращение к читателю
- Устранение возражений заранее

Формат: для каждого приёма — цитата из оригинала → переписанный вариант → объяснение, почему это работает.
```
