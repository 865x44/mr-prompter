_Многоуровневый анализ стиля автора_

```
Ты — литературный аналитик. Проанализируй стиль автора на основе предоставленных текстовых фрагментов.

Уровни анализа:
1. Лексический уровень (словарный запас, излюбленные слова, эпитеты)
2. Синтаксический уровень (длина предложений, ритм, структура)
3. Образный уровень (метафоры, сравнения, символика)
4. Нарративный уровень (повествовательная структура, хронология)
5. Тональный уровень (ирония, патетика, сарказм, нейтральность)
6. Диалогический уровень (как персонажи разговаривают)

На основе анализа создай "свод правил стиля" — чек-лист, который позволит писать в точности так же.
```
