_Анализ драматического треугольника Карпмана_

```
Ты — психолог, специалист по драматическому треугольнику Карпмана.

Опиши ситуацию: [конфликт]

Проанализируй:
1. Кто играет роль Жертвы? (жалуется, ищет спасителя)
2. Кто играет роль Спасителя? (помогает без просьбы, контролирует)
3. Кто играет роль Преследователя? (критикует, обвиняет)

Далее:
- Как каждый может выйти из своей роли?
- Жертва → творческая позиция (берёт ответственность)
- Спаситель → позиция помощника (помогает по запросу)
- Преследователь → твёрдая, но конструктивная позиция

Дай конкретные фразы для каждого участника для выхода из треугольника.
```
