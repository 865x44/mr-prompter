### style_cloning_parallel_texts
**Категория:** chain | **Модель:** any | **Язык:** ru
_Клонирование стиля через параллельные тексты_

```
Для клонирования стиля переводчика:
1. Загрузи оригинал на английском + перевод на русском (30-40 стр.)
2. Создай универсальные директивы для определения портрета переводчика
3. По директивам создай уникальный портрет переводчика
4. Примени портрет для перевода нового текста

Важно: сначала создай глоссарий имён/названий для консистентности.
```
