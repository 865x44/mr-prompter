### if_when_pattern
**Категория:** template | **Модель:** any | **Язык:** ru
_Паттерн "Когда" вместо "Если"_

```
Правило: заменяй "Если" на "Когда" для повышения уверенности инструкции.

❌ "Если в ответе добавишь списки, укажи..."
✅ "Когда в ответе добавишь списки, укажи..."

❌ "Если найдёшь ошибки, сообщи..."
✅ "Когда найдёшь ошибки (а ты их найдёшь), сообщи..."
```
